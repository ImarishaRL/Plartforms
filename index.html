Imarisha

A project to create Reinforcement Learning Plartforms for existing African board games.

About Imarisha

Imarisha aims solve abstract strategy African board games using reinforcement learning, this by creating plartforms and encouraging reserchers to venture into game theory and multi-agent reinforcement learning using this boards boardgames.

Open source Yes
Language Python
Game diversity African boardgames
Game customization Yes
Game Intergration Tightly
